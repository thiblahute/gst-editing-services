Instance description

Outputs the video stream from a given image sequence. The start frame chosen will be determined by the in-point property on the track element.

Properties

The “uri” property 

“uri” unicode

The uri of the file/resource to use. You can set a start index, a stop index and a sequence pattern. The format is <multifile://start:stop_location_-pattern>. The pattern uses printf string formating.

Example uris:

multifile:///home/you/image%03d.jpg

multifile://20:50@/home/you/sequence/%04d.png

Flags: Read / Write / Construct Only
